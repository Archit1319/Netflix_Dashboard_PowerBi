# 🎬 Netflix Content Analysis Dashboard (Power BI)

![Netflix Logo](https://upload.wikimedia.org/wikipedia/commons/0/08/Netflix_2015_logo.svg)

## 📌 Project Overview
The **Netflix Content Analysis Dashboard** is a powerful data visualization project built using **Power BI**. It provides an in-depth dive into the Netflix library, helping stakeholders understand content trends, audience preferences, and global production patterns.

Whether you're looking for the growth of original content or the distribution of ratings across different countries, this dashboard offers actionable insights at a glance.

---

## 🚀 Key Features
- **Content Distribution:** Comprehensive comparison between Movies and TV Shows.
- **Timeline Trends:** Yearly growth of content additions from 2008 to 2021.
- **Global Footprint:** Heatmaps and charts showing top-producing countries.
- **Genre Analysis:** Deep dive into popular categories like Dramas, Comedies, and Documentaries.
- **Rating Breakdown:** Visualization of content maturity levels (TV-MA, TV-14, PG-13, etc.).
- **Duration Insights:** Analysis of movie runtimes and TV show season counts.

## 📖 Data Description
The analysis is based on a dataset containing over **6,000+ titles**, featuring:
- **Show ID:** Unique identifier for each title.
- **Type:** Movie or TV Show.
- **Director & Cast:** Information about the talent behind the content.
- **Release Year:** The original year the content was released.
- **Rating:** Content maturity level.
- **Listed In:** Detailed categories/genres.

## ⚙️ How to Run
1. **Prerequisites:** Install [Power BI Desktop](https://powerbi.microsoft.com/desktop/).
2. **Setup:**
   - Clone this repository or download the ZIP.
   - Navigate to the `Dashboard` folder.
3. **Execution:**
   - Open `Netflix DashBoard.pbix`.
   - If prompted for a data source, point it to the CSV files in the `Data Set` folder.
